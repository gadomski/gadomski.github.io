Real-time glacier monitoring

Sep 10, 2014  

One of my work projects went live today: http://glacierresearch.org/.

Internally known as glac.io (and also available at http://glac.io/), the website serves real-time climate and camera data from sensors located near glaciers all over the world. Some of the highlights:

  • Real time climate data charting: constructed with d3.js and a custom-rolled javascript api for the Corps Water Management System. These data provide up-to-the-minute information about the environmental conditions at each of three glacier sites (with more to come soon).
  • Real time images: we currently have four cameras hooked into our site providing recent still imagery of our research sites. I’ve made a viewer (again, with d3.js) to flip through the latest pictures. Some of these pictures can be stunning.

  • Room for growth: the webpage is developed with an eye towards incorporating data from more sites and more projects. The code is available on github and uses all open-source technologies. The site itself is all static html pages and javascript — no complex server setup necessary.

Many agencies and individuals have worked together to bring these data to one place, and I owe them all a thanks.
